# Flores-do-campo

Este é um site simples criado para apresentar diferentes tipos de flores.  
O projeto foi desenvolvido como parte de um exercício de prática usando GitHub Pages e HTML/CSS básico.

- Estilização com CSS
- Tema floral 🌷
